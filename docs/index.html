Features

🎲

Monte Carlo simulation

Vectorised NumPy engine. 10,000 runs complete in under a second.

📊

Sensitivity analysis

Spearman rank correlation identifies which variables drive outcomes most.

🔒

Fully offline

No network calls. No API keys. Works without any external service.

🧩

Extensible scenarios

Add a new scenario by subclassing BaseScenario and registering it in the engine.

🤖

Optional local AI

Pass --explain to get a plain-text analysis from a local Ollama model.

📈

Charts

Pass --chart to save a dark-themed score histogram as a PNG.

Installation

git clone https://github.com/NarimaneBr/path-sim
cd path-sim
pip install -e .

# With dev tools
pip install -e ".[dev]"

# With LLM support (requires Ollama)
pip install -e ".[llm]"

Usage

# Named scenarios
pathsim startup
pathsim career-change
pathsim investment

# Natural language
pathsim "start a startup"

# With overrides
pathsim startup --runs 20000 --capital 500000 --team 4 --experience high

# Export chart
pathsim startup --chart --chart-path results/startup.png

# Local LLM explanation (Ollama required)
pathsim startup --explain --model mistral

# List available scenarios
pathsim --list

Design

Each simulation samples independent variables from calibrated probability distributions, combines them through a weighted linear model, and classifies the result. Running thousands of iterations produces a stable outcome distribution.

Scoring model (startup)

score =
    0.30 × founder_skill
  + 0.25 × market_size
  + 0.20 × market_timing
  + 0.15 × capital
  + 0.10 × competition_inverse

Outcome thresholds

ScoreClassification
≥ 0.65Success
0.35 – 0.65Moderate outcome
< 0.35Failure

Weights and thresholds are calibrated by hand, not fitted to real outcome data. Treat them as a starting point for exploration, not ground truth.
